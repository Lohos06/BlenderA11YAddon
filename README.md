# BlenderA11YAddon

## Parcour Utilisateur 

L'utilisateur arrive sur blender, va dans les paramètres configures l'ajout de l'extension.
Une fois notre add on ajouté, il doit soit ouvrir sur le bord à droite juste à côté du repère des axe XYZ soit appuyé sur N.
Une fois ouvert, il doit choisir l'onglet vertical nommé SIMPLE . 
Il pourra ensuite cliqué sur les différents bouttons selon ses besoins: 

- button 1 : enlever le header
- button 2 : créer un nouveau workplace
- button 3 : enlever tout les autres workplaces
